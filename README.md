# ⚡ LevelUp AI — Autonomous Skill-to-Internship Agent OS

**LevelUp AI** is a highly immersive, gamified web application built using **FastAPI** and **Streamlit** that helps candidate job seekers transition from learning and timed proctored testing to landing jobs using fully autonomous AI career growth agents.

The system features a **premium Cyberpunk Neon Dark-Mode SaaS UI** tailored for high interaction and stunning micro-animations.

---

## 🚀 Key Product Features

1. **🔌 Multi-Mode AI Engine Selector**
   - **Cloud API Engines**: OpenAI (`gpt-4o-mini`), Gemini (`gemini-1.5-flash`).
   - **Offline Local Inference**: Ollama-based models (`llama3`, `mistral`, `deepseek-coder`, `codellama`, `phi3`) communicating over local REST APIs.
   - **Intelligent Simulated Fallback**: Play and test immediately without any external API keys or Ollama setup using our contextual AI simulator!

2. **🛡️ Secure Proctoring & Adaptive Assessments**
   - Floating circular HUD proctor widgets utilizing HTML5 user-camera capture with eye-focus tracking scan sweeps.
   - Active countdown timer submitting answers instantly on timeout.
   - Visibility API tracking tab switches and window blur events with audio alarms, alerting popups, and instant lockdown limits.
   - Right-click, select, copy, and paste disabled on exam sheets to maintain integrity.

3. **📊 Dynamic Skill Radar Plotting**
   - Candidate stats automatically generated by scanning uploaded resumes (using `pypdf` and LLM parsing).
   - Dynamic polar radar charts mapping candidate verified technical strengths and target improvement ratings.

4. **🏋️ Gamified Hands-on Learning Hub**
   - Weak skills are automatically converted into coding practice tasks.
   - Integrated logic sandbox allowing script submission and immediate AI Review Debugger feedback.
   - Successfully solving tasks raises validated skill ratings and grants XP levels!

5. **🤖 Autonomous Auto-Apply Headhunter Agent**
   - Ranks-based system unlocks:
     - **Novice Rank (Score < 70%)**: Locked. Confined to sandbox study hub tasks.
     - **Advanced Rank (Score 70–89%)**: Unlocks limited applications (1-2 per day) and custom cover letter composition.
     - **Elite Rank (Score >= 90%)**: Fully unlocks the Autonomous Auto-Apply Bot, composing personalized metrics-driven ATS optimized resumes, cover letters, and running the submissions with step-by-step rolling cyberpunk console log sweeps!

## 🛠️ Step-by-Step Launch Guide

### 1. Prerequisite Installations
Ensure Python 3.9+ is installed on your computer.

### 2. Install Project Dependencies
Open your shell terminal in the project root directory and execute the unified root installation:
```bash
pip install -r requirements.txt
```

### 3. (Optional) Run Local Offline Models
If you want to use Ollama offline local models:
- Download and run [Ollama](https://ollama.com).
- Open your terminal and pull a model:
```bash
ollama pull llama3
```

### 4. Run the System Application
To launch the entire platform (both backend Uvicorn and Streamlit front-end UI concurrently), run:
```bash
python run.py
```

The system will start FastAPI on `http://127.0.0.1:8000` and automatically load the cyberpunk dashboard on `http://localhost:8501`.
Enjoy your journey to becoming a verified career agent! 🚀
